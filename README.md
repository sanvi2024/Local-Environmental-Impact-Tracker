**Local-Environmental-Impact-Tracker**

This is a simple, console-based Java application built to track basic digital and physical activities to estimate a user's carbon footprint.

**Description**

This program allows you to:
Log your digital activities (hours of device usage).
Log your physical transportation (kilometers by car, bike, or walking).
Based on these logs, it provides a very basic estimate of your carbon footprint in kg CO2 and offers simple suggestions for greener alternatives.

**Features**

Log device usage
Log transportation
Show estimated carbon footprint
Get simple suggestions
